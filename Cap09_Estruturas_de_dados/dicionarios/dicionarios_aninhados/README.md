Dicionaários Aninhados(nested dicionary)
=====================

Em Python, um dicionário aninhado é um dicionário dentro de outro dicionário.

É uma coleção de dicionários em um único dicionário.

Fonte: https://www.programiz.com/python-programming/nested-dictionary
